# Wheat Futures Price Analysis Project
Author: Weitong Niu  
University of Utah, B.S. in Applied Mathematics

## Project Overview
This project analyzes wheat historical prices, import/export data, and macroeconomic indicators to build a predictive model for wheat futures pricing. The goal is to identify key factors influencing price fluctuations and generate data-driven forecasts using regression models and data visualization.

## Techniques Used
- Python data processing & cleaning
- Data visualization & exploratory analysis
- Regression modeling & predictive analysis
- Feature engineering & statistical analysis

## Skills Demonstrated
- Data Science
- Quantitative Modeling
- Financial Data Analysis
- Python Programming
- Business Application of Data Science
